How It Works

1

Input Keywords

Feed in your keyword list. Nichedigger scores each one with 18 intent types and brand fitness.

3

Searches Reddit via JSON API. Extracts buying signals, pain points, and competitor mentions. 3-round iterative deepening.

Every post goes through a topical relevance check. Irrelevant results get zero signals. No false positives.

5

Priority Ranking

P0-P3 based on: commercial intent (45%), Reddit live signals (25%), volume (20%), KD penalty (10%). KD>60 capped at P1.

6

Output

JSON report + CSV + this dashboard. See why each keyword got its priority, backed by real Reddit evidence.
